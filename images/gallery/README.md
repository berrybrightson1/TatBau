# Gallery images

Add 6 images here as **gallery-1.jpg** … **gallery-6.jpg**, then in `src/app/[locale]/galerie/page.tsx` set `GALLERY_IMAGES` to use these paths:

```ts
const GALLERY_IMAGES = [
  { src: "/images/gallery/gallery-1.jpg", altKey: "door_1" },
  // … through door_6
];
```
